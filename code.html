Introduction

The Code is structured in three separate functions. Each function has its own responsibilities which makes it easier to maintain and update the code base through it's life cycle. One function acts as our controller, the other our model or logic, and the last one functions as our view or display code.

The getValue() function

The first statement in the getValue() function gets the element with an id of "alert" and adds the classList.Add property and method of invisible. This hides the message until we call the logic function of reversedString and the view function displayString.

The reversedString() function

In this function we grab the input from the user which is passed as an argument. Next, a for loop is created. Inside the for loop the index is initialized by the length of the string. The conditional is then set to have the index greater than or equal to zero. Lastly, the index is decremented until the loop reaches zero.

The displayString() function

Here we get the element with an id of message, using a template string we created our display message. The second statement is required to remove the invisilble class from the element with the alert id. This will allow our message to be displayed.

The clearForm() function

This function clears the input field and the alert message after the Reset button is selected. An empty string is assigned to the clearString array. Next, we assign the clearString array to the document.getElementById("userString").value expression.
